Code Block Component for FPM Packages

Package Name
fifthtry.github.io/code-block
This package can be used to show source code in any document.
Output on browser will as below:

How to add: onto your project:

Add this to FPM.ftd
-- fpm.dependency: fifthtry.github.io/code-block as cb

-- fpm.dependency: fifthtry.github.io/code-block as cb
Then use the component:
-- fpm.dependency: fifthtry.github.io/code-block as cb

-- import: cb

-- cb.code:
if: $caption is not null
caption: $caption
lang: $lang

$body

-- fpm.dependency: fifthtry.github.io/code-block as cb

-- import: cb

-- cb.code:
if: $caption is not null
caption: $caption
lang: $lang

$body

  • lang is the file name extension.

  • caption is the the title/caption of code block.

  • lang is the file name extension.

  • body is main body of the code block.

Usage: how to write

How to write code block of image shown above
-- cb.code: hello world ftd
lang: ftd

-- string msg: hello world
-- ftd.text: $msg

-- cb.code: hello world ftd
lang: ftd

-- string msg: hello world
-- ftd.text: $msg

Output: of above code snippet

hello world ftd
-- string msg: hello world
-- ftd.text: $msg

-- string msg: hello world
-- ftd.text: $msg
